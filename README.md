# 3D STG (Docs Only)

Three.js/Vite/TypeScriptで作るスペースハリアー風3Dシューティングの設計ドキュメント置き場です。ソースコードは含まれていません。

## 構成
- `Docs/REQUIREMENTS.md`: 要件定義
- `Docs/IMPLEMENTATION_PLAN.md`: 実装計画
- `Docs/TEST_PLAN.md`: テスト計画

## 予定されている機能概要
- フェーズ1-2まで実装済み想定（プレイヤー移動・射撃、敵3種、ステージ/スコア、背景/天井、敵弾）
- 今後の拡張: ステージ4中ボス、ステージ5蛇型ボス、UI改善、サウンド

## ビルド/デプロイ想定
Viteを利用し、GitHub Pagesで公開する想定です。

```bash
npm run build
npm run deploy
```
